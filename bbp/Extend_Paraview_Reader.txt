------ ParaView External Modules -------------

http://www.cmake.org/Wiki/ParaView:Extend#Include_Custom_Class_During_Compile_Time

import the code to the ParaView build process using ParaView External Modules

ParaView External Modules are collections of code that are included during the ParaView build process, but reside somewhere outside the ParaView source tree.

To generate a ParaView External Module, we need to create a file called <Module-Name>ParaViewImport.cmake.
